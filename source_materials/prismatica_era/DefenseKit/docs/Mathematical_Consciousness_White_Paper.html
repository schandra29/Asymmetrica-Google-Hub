2.2 Three-Regime Clustering Framework

Our core hypothesis posits that mathematical structures naturally organize into three distinct operational regimes:

  • Exploration (R1): Characterized by high variance, novel pattern discovery, and boundary testing
  • Optimization (R2): Marked by refinement processes, local improvements, and efficiency maximization
  • Integration (R3): Distinguished by stability, pattern consolidation, and equilibrium maintenance

2.2.1 Statistical Validation Protocol

For each domain, we employed the following validation sequence:

  1. Feature Engineering: Domain-specific mathematical properties extraction
  2. Clustering Analysis: K-means clustering with k=3 (pre-registered parameter)
  3. Statistical Testing: ANOVA for regime separation (α = 0.001)
  4. Effect Size Calculation: Eta-squared (η²) with threshold > 0.5
  5. Bootstrap Validation: 100 iterations for stability assessment
  6. Cross-Validation: 5-fold stratified validation

Appendix A: Statistical Methodologies

A.1 Clustering Validation Metrics

For each domain, clustering quality was assessed using multiple metrics:

  • Silhouette Score: Measures cluster cohesion and separation
  • Calinski-Harabasz Index: Evaluates cluster variance ratios
  • Adjusted Rand Index: Assesses clustering stability across bootstrap samples
  • Within-Cluster Sum of Squares: Measures cluster compactness

A.2 Cross-Validation Protocol

All results were validated using stratified 5-fold cross-validation with the following protocol:
